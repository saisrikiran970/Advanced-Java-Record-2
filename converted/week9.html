Subject: Advanced Java Lab (23IT5351)

Topic: REST API using Spring Boot

AIM

To develop a REST API using Spring Boot and test GET and POST requests using Postman.

Procedure

1. Create a Spring Boot project using Spring Initializr.

2. Add dependency: Spring Web.

3. Import the project into STS / IntelliJ IDE.

4. Create a REST controller and define GET and POST mappings.

5. Run the application (default port: 8080).

HelloController.java

package com.example.demo;

import org.springframework.web.bind.annotation.*;

@RestController
public class HelloController {

    @GetMapping("/hello")
    public String sayHello(@RequestParam String name) {
        return "Hello, " + name + "!";
    }

    @PostMapping("/add")
    public String addNumbers(@RequestBody Numbers num) {
        int sum = num.a + num.b;
        return "Sum of numbers: " + sum;
    }
}

class Numbers {
    public int a;
    public int b;
}

Testing REST APIs using Postman

GET API

URL: http://localhost:8080/hello?name=Sai

Output: Hello, Sai!

POST API

URL: http://localhost:8080/add

Request Body (JSON):

{
  "a": 10,
  "b": 20
}
Output: Sum of numbers: 30

Result

REST APIs using GET and POST methods were successfully implemented and tested using Spring Boot and Postman.
