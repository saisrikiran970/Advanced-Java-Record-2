1. Difference between Core Java and Advanced Java

Core Java

Core Java contains the fundamental concepts of Java such as syntax, object-oriented programming, and standard libraries. It forms the foundation of Java programming.

Key Features:
  • Basic Language Constructs
  • Object-Oriented Programming (OOP)
  • Standard Libraries
  • Exception Handling
  • Multithreading and Concurrency
  • Basic I/O Operations

Advanced Java

Advanced Java builds on Core Java and focuses on enterprise, web-based, and distributed applications.

Key Features:
  • Database Connectivity (JDBC)
  • Web Technologies
  • Enterprise Java
  • Web Services
  • Advanced Networking
  • Security

2. Important Features of Java Applets

A Java Applet is a Java program that runs inside a web browser. It is embedded in an HTML page using <applet> or <object> tags. Applets run in a secure sandbox environment.

Key Features:
  • Interactive and Dynamic Content
  • Graphical Capabilities
  • Platform Independence
  • Event Handling
  • Limited System Access
  • Embedding in HTML
  • Parameterization
  • Browser Interaction
  • Security Restrictions

3. Key Features of Java Servlets

A Java Servlet is a server-side Java program that handles client requests, processes them, and generates dynamic responses.

Key Features:
  • Platform Independent
  • Server-Side Processing
  • HTTP Protocol Support
  • High Performance
  • Robustness and Security
  • Session Management
  • Integration with Java Ecosystem
  • Extensibility
  • Dynamic Content Generation

4. Differences between Applets and Servlets

Applets

  • Run on the client-side (browser)
  • Used for GUI-based interaction
  • Restricted by browser security
  • Lifecycle: init(), start(), stop(), paint(), destroy()

Servlets

  • Run on the server-side
  • Generate dynamic web content
  • No GUI support
  • Lifecycle: init(), service(), destroy()
  • Communicate using HTTP requests and responses

6. Role of JDBC and ODBC

JDBC enables Java applications to connect and interact with databases. ODBC is a language-independent API used by various programming languages.

7. Difference between JDBC and ODBC

JDBC is Java-specific, while ODBC supports multiple programming languages like C and C++.

8. Difference between Cookies and Sessions

Cookies are stored on the client browser and are less secure. Sessions are stored on the server and provide better security.

9. Spring Boot vs Traditional Spring

Spring Boot simplifies development using auto-configuration and starters, while traditional Spring requires extensive manual configuration.

10. Standalone vs Web Applications

Standalone applications run locally on a system, while web applications run on servers and are accessed via browsers.

11. Basic Requirements for a Web Application
